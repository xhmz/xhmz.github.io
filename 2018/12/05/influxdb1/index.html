Influxdb Introduction

xhmz's blog
2018-12-05

Introduction

InfluxDB is an open source time series database with no external dependencies.
It’s useful for recording metrics, events, and performing analytics.

Github

https://github.com/influxdata/influxdb

基本概念

  • database：数据库
  • measurement：you can think of a measurement as an SQL table, where the primary index is always time.
  • tag：index，类似于索引，key value，都是字符串
  • field：字段
  • timestamps：每一行记录都可以增加一个timestamps，如果没有指定，会使用服务器当前时间，纳秒级别

基本语法

INSERT

insert <measurement>[,<tag-key>=<tag-value>…] <field-key>=<field-value>[,<field2-key>=<field2-value>…] [unix-nano-timestamp]
比如：tables,tag1=1,tag2=2 field1=1,field2=2

insert tables,tag1=1,tag2=2 field1=1,field2=2
SELECT

SELECT<field_key>[,<field_key>,<tag_key>] FROM <measurement_name>[,<measurement_name>]

select * from tables where "tag1"='1'

敲黑板 根据 tag 查询 where的语法 tag key必须用双引号，tag value必须为 单引号
